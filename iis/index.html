Quick Start

  1. Place this index.html into your IIS site’s root (usually C:\\inetpub\\wwwroot).
  2. Open Internet Information Services (IIS) Manager → select your site → Browse ":80".
  3. Replace this page with your application when ready. Keep it as a friendly 200 OK health check.
